# MDX Showcase

This page demonstrates the various capabilities of MDX files, which combine Markdown with JSX.

## Basic Markdown Features

### Text Formatting
You can write text with **bold**, *italic*, or ***both***. 
You can also use ~~strikethrough~~ and `inline code`.

### Lists
Unordered list:
- Item 1
- Item 2
  - Nested item
  - Another nested item
- Item 3

Numbered list:
1. First item
2. Second item
3. Third item

### Blockquotes
> This is a blockquote
> It can span multiple lines
>> And can be nested

### Code Blocks
```python
def hello_world():
    print("Hello, World!")
```

### Tables
| Feature | Supported | Notes |
|---------|-----------|-------|
| Tables | ✅ | With alignment |
| Lists | ✅ | Nested too |
| Code | ✅ | With syntax highlighting |
